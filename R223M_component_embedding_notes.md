## Principle

Components must not appear as a tool shelf in the teacher-facing draft. They
are embedded as classroom actions inside the teaching process.

## Embedded Components

| Component | Embedded As | Draft Location | Why It Is Not A Card Wall |
| --- | --- | --- | --- |
| 圈一圈 / circle_and_annotate | Teacher models marking key structure on screen | 六（二）观察发现 | It appears as an observation method, not a separate tool panel |
| 技法菜单 / technique_menu | Materials and techniques listed as optional creation support | 六（四）拓展创作 | It is part of teacher preparation and student choice |
| 学习单记录 / worksheet_record | Four learner-facing fields | 五、教学准备; 七、评价与证据 | It preserves evidence, not an isolated component |
| 关键词评价 / keyword_feedback | Four evaluation words: structure, function, material, use | 六（五）笔友汇交流 | It supports student speech and peer review |
| 拍照提交 / photo evidence | Optional process evidence | 七、评价与证据 | It is a record method, not a classroom execution surface |

## Misuse Guard

```text
Do not list 12 components.
Do not make component names visually louder than the lesson flow.
Do not imply live classroom execution.
Do not put component configuration in the main teacher draft.
```
